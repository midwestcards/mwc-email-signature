Raw HTML
Gmail: Settings (gear) → See all settings → General → Signature → Create new → click the signature box and paste (⌘/Ctrl-V) → Save changes.
Outlook (web / new): Settings → Mail → Compose and reply → paste into the signature editor → Save.
Outlook (Windows desktop): Use Show HTML → Copy code, or paste the rendered signature directly into File → Options → Mail → Signatures.

Brand assets

The logo, Insider banner, and all social/contact icons are hosted in this repo’s assets/ folder and wired in via the CONFIG block at the top of index.html. To swap one, replace the file in assets/ keeping the same filename and push — GitHub Pages redeploys automatically. Never rename or delete files there: every signature already in use points at those URLs.
