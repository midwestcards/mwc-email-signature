Live preview renders as it will in email
Raw HTML
Gmail: Settings (gear) → See all settings → General → Signature → Create new → click the signature box and paste (⌘/Ctrl-V) → Save changes.
Outlook (web / new): Settings → Mail → Compose and reply → paste into the signature editor → Save.
Outlook (Windows desktop): Use Show HTML → Copy code, or paste the rendered signature directly into File → Options → Mail → Signatures.

Brand assets

The stacked Midwest Cards logo is already wired in (hosted in this repo’s assets/ folder). To add the Insider banner (330px wide), drop a PNG into assets/, set its URL in the CONFIG block at the top of index.html, and push — GitHub Pages redeploys automatically and everyone gets it. PNG only (SVG won’t render in Gmail/Outlook).
